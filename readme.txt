== Description ==

Monetize your WordPress website/blog with XIP.ir shortlinks. This plugin converts your links to XIP.ir links automatically.

You will able to:

*   Enable/Disable XIP.ir feature
*   Convert either all links or only outgoing links to XIP.ir
*   Choose ad type: Interstitial or Banner ad

    You will need an XIP.ir id. To get it, login to XIP.ir go to http://XIP.ir/referrals.php and find ?r=xxxx. 
xxxx is your XIP.ir ID.

== Installation ==

1. Go to Plugins-> Add New Plugin -> Upload
2. Upload `xip-wordpress-plugin.zip`
3. Activate the plugin.
4. Navigate to Settings-> XIP.ir Settings, enable plugin and put your XIP.ir id. To get it, login to XIP.ir go to http://XIP.ir/referrals.php and find ?r=xxxx. 
xxxx is your XIP.ir ID.
